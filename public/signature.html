READ THIS FIRST:
Before pasting the new signature, you MUST clear out the old one first.

Step 1: Open Outlook signature editor. Click in the signature box → press Cmd+A / Ctrl+A → press Delete. The box must be completely empty.
Step 2: Come back to this page and copy/paste the new signature below.
How to install (60 seconds):
1. Make sure the Outlook signature box is empty (see above).
2. Click anywhere inside the white box below.
3. Press Cmd+A / Ctrl+A to select all.
4. Press Cmd+C / Ctrl+C to copy.
5. Click into the empty Outlook signature box → paste with Cmd+V / Ctrl+V.
6. Tick "Set default for new messages" and "Set default for replies and forwards".
7. Hit Save.
